## 常用公式

### 杠杆计算

```
仓位价值 = 保证金 × 杠杆倍数
```

**例子：**
保证金：$1,000  
杠杆：10倍  
仓位价值：$1,000 × 10 = $10,000  

### 爆仓价格

```
做多爆仓价 = 开仓价 × (1 - 1/杠杆倍数)
做空爆仓价 = 开仓价 × (1 + 1/杠杆倍数)
```

**例子（10倍杠杆）：**
开仓价：$100,000  
做多爆仓价：$100,000 × (1 - 1/10) = $90,000  
做空爆仓价：$100,000 × (1 + 1/10) = $110,000  

### 收益率计算

```
收益率 = (平仓价 - 开仓价) / 开仓价 × 杠杆倍数
```

**例子（10倍杠杆做多）：**
开仓价：$100,000  
平仓价：$105,000  
涨幅：5%  
收益率：5% × 10 = 50%  

### 资金费率计算

```
每8小时支付 = 持仓价值 × 资金费率
```

**例子：**
持仓价值：$100,000  
资金费率：+0.01%  
支付：$100,000 × 0.01% = $10

## 时区转换

### UTC时间（交易所常用）

UTC 8:00 = 北京时间 16:00  
UTC 16:00 = 北京时间 00:00（次日）
