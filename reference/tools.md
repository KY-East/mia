# 推荐工具速查

## 📱 交易所

### Binance (币安)
- **官网：**[binance.com](https://www.binance.com)
- **用途：**现货+合约，综合交易
- **优点：**流动性最好，币种最全
- **手续费：**0.1%（现货）

### Bitget
- **官网：**[bitget.com](https://www.bitget.com)
- **用途：**合约交易、跟单交易
- **优点：**跟单功能领先，合约体验好
- **手续费：**0.085%（现货）

### Bybit
- **官网：**[bybit.com](https://www.bybit.com)
- **用途：**合约交易
- **优点：**新手友好，客服好
- **手续费：**0.1%（现货）

## 💼 钱包

### Phantom
- **官网：**[phantom.app](https://phantom.app)
- **用途：**Solana链专用钱包
- **优点：**界面友好，速度快，Gas费便宜
- **适合：**买Solana链上的Meme币

### MetaMask
- **官网：**[metamask.io](https://metamask.io)
- **用途：**EVM链通用钱包（以太坊、BSC等）
- **优点：**兼容性最好，生态完善
- **适合：**DeFi、以太坊生态

### Bitget Wallet
- **官网：**Bitget App内置
- **用途：**多链钱包
- **优点：**支持多链，CEX和链上无缝切换
- **适合：**多链玩家

## ⚠️ 工具使用注意事项

### 下载渠道

✅ **正确：**
- 官方网站
- App Store / Google Play
- Chrome Web Store

❌ **不要：**
- 第三方网站
- 不明来源的安装包
- 盗版软件

### 授权管理

✅ **定期检查：**
- 每月检查一次钱包授权
- 撤销不用的授权
- 只授权必要的权限

❌ **不要：**
- 无限授权（Unlimited Approval）
- 授权给不明合约
- 长期不检查

### 数据安全

✅ **注意：**
- 不要在工具里保存私钥
- 使用只读模式
- 定期清除缓存
