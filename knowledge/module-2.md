# 模块2: 链上操作与Meme币狩猎

## 2.1 Web3钱包 - 你的数字保险柜（45分钟）

### CEX vs 链上钱包 - 托管 vs 自主

**中心化交易所钱包（CEX）：**

**本质：**
你的币存在交易所的数据库里
交易所帮你保管

**类比：**
把钱存在银行
- 银行控制你的钱
- 你只是有存款凭证
- 银行倒闭，你的钱可能没了


✅ 方便（不用管私钥）
✅ 丢密码可以找回
✅ 有客服支持
✅ 交易快速（内部转账）


⚠️ 不是真正的"拥有"
⚠️ 交易所跑路，你的钱没了
⚠️ 交易所被黑客攻击，币可能被盗
⚠️ 账户可能被冻结

**风险案例：**
- 2022年FTX倒闭：用户损失数十亿美元
- 2014年Mt.Gox被盗：85万个BTC丢失

**链上钱包（Web3 Wallet）：**

**本质：**
你100%拥有和控制
只有你有私钥/助记词

**类比：**
把现金放在自己的保险柜
- 只有你知道密码
- 别人无法动你的钱
- 但丢了密码，神仙也救不了你


✅ 真正的拥有（去中心化）
✅ 任何人无法冻结你的账户
✅ 交易所倒闭也不影响你
✅ 可以参与DeFi、买Meme币、打新


⚠️ 私钥丢了=永久损失（无法找回）
⚠️ 发错地址=永久损失
⚠️ 被钓鱼=永久损失
⚠️ 没有客服（一切靠自己）

**风险：**
全部由你自己承担

**什么时候用CEX，什么时候用链上钱包？**

CEX更适合
✅ 大额资金交易
✅ 做合约
✅ 长期持有（但要选可靠的交易所）
✅ 新手（容错率高）

链上钱包更适合
✅ 买Meme币（DEX上的币）
✅ 参与DeFi（挖矿、借贷）
✅ 参与IDO/IEO打新
✅ 需要完全自主控制

**Mia建议：**
两个都要有，分散风险
- 大钱（>$5000）放CEX
- 小钱（几百刀）玩链上

### 创建Phantom钱包 - 完整SOP（15分钟）

**Step 1: 下载安装（3分钟）**

**电脑端（浏览器插件）：**
1. 打开Chrome浏览器
2. 访问 phantom.app
3. 点击"Download"
4. 选择"Chrome Extension"
5. 跳转到Chrome Web Store
6. 点击"添加至Chrome"
7. 确认"添加扩展程序"
8. 安装完成

**标志：**
浏览器右上角出现紫色幽灵图标

**手机端（App）：**

**iOS：**
1. 打开App Store
2. 搜索"Phantom"
3. 找到紫色幽灵图标的App
4. 点击"获取"
5. 下载安装

**Android：**
1. 打开Google Play
2. 搜索"Phantom"
3. 下载安装

⚠️ **注意：**
确认开发者是"Phantom"
防止下载到假App

**Step 2: 创建新钱包（5分钟）**

**2.1 打开Phantom：**
点击浏览器右上角的幽灵图标
（或打开手机App）

看到欢迎页面：
```
┌──────────────────────┐
│   👻 Phantom         │
│                      │
│  A crypto wallet     │
│  reimagined          │
│                      │
│  [Create New Wallet] │
│  [Import Wallet]     │
└──────────────────────┘
```

**2.2 选择创建新钱包：**
点击"Create New Wallet"

⚠️ 不要点"Import Wallet"
那是用来导入已有钱包的

**2.3 设置密码：**

**输入密码要求：**
- 至少8位
- 建议：大写+小写+数字+符号

例如：Phantom2024!

**这个密码用于：**
- 在这台设备上解锁钱包
- 注意：这不是恢复钱包的密钥！
- 换设备不需要这个密码

输入密码
确认密码
勾选"我已阅读并同意服务条款"
点击"Continue"

**Step 3: 备份助记词（⚠️⚠️⚠️ 最最最重要！）**

**3.1 什么是助记词？**

助记词 = 12个英文单词

例如：
```
apple banana cat dog elephant
fish garden house island juice
king lemon
```

**这12个词就是你的钱包！**
- 丢了 = 钱包永久丢失
- 泄露 = 别人可以盗走你的币
- 只要有这12个词，可以在任何设备恢复钱包

**重要性：**
助记词 > 密码
助记词 > 你的命（开玩笑，但差不多）

**3.2 查看助记词：**

Phantom显示：
```
┌────────────────────────┐
│ Secret Recovery Phrase │
│                        │
│ 1. apple               │
│ 2. banana              │
│ 3. cat                 │
│ 4. dog                 │
│ ... (共12个词)         │
│                        │
│ ⚠️ 永远不要分享给任何人 │
│                        │
│ [I saved my phrase]    │
└────────────────────────┘
```

**3.3 备份助记词（必须做的！）：**

**备份方式1：纸笔抄写（最安全！）**
1. 拿3张纸
2. 用笔抄下这12个词
3. 必须按顺序！（1-12）
4. 检查拼写（一个字母都不能错）
5. 抄3份，放在不同地方：
   - 第1份：家里保险柜
   - 第2份：父母家/其他安全地点
   - 第3份：银行保险箱（如有）

**备份方式2：钢板备份（终极安全）**
1. 在亚马逊买"Crypto Steel"
2. 把助记词刻在钢板上
3. 防火防水防盗
4. 成本$30-100
5. 适合大额资产

**备份方式3：加密存云端（可选）**
1. 把助记词输入文本文件
2. 用强密码加密（AES-256）
3. 上传到云盘
4. 密码不要和助记词放一起

**3.4 绝对不能做的事！**

❌ 截图保存（手机被黑客入侵会泄露）
❌ 发给任何人（包括自己的其他设备、朋友、家人）
❌ 输入到任何网站（除了恢复钱包）
❌ 存在未加密的云盘
❌ 只抄一份（万一丢了就完了）
❌ 发给Phantom客服（真客服永远不会要你的助记词）
❌ 用邮件发给自己

**真实案例：**

**案例1：**
某用户把助记词截图存在手机相册
→ 手机中病毒
→ 黑客窃取相册
→ 钱包里$50,000被盗光
→ 无法追回

**案例2：**
某用户只抄了一份助记词
→ 放在钱包里
→ 钱包丢了
→ 助记词永久丢失
→ 钱包里的币永久锁死
→ 无法取回

**3.5 验证备份：**

点击"I saved my phrase"后：


**如果点错：**
说明你抄错了或没记住
必须重新查看助记词，重新抄

## 2.2 去中心化交易所(DEX) - 链上换币实操（30分钟）

### DEX vs CEX - 自动售货机 vs 银行

**中心化交易所（CEX）：**

**运作方式：**
- 有公司运营（Binance、Bitget）
- 订单撮合系统：
  * 买家下单买BTC $100,000
  * 卖家下单卖BTC $100,000
  * 系统撮合成交
- 资金托管在交易所


✅ 流动性好（买卖快）
✅ 价格稳定
✅ 有客服支持
✅ 界面友好


⚠️ 需要KYC（实名）
⚠️ 可能跑路
⚠️ 账户可能被冻结
⚠️ 新币上线慢（需要审核）

**去中心化交易所（DEX）：**

**运作方式：**
- 没有公司，智能合约自动运行
- 自动做市商（AMM）机制：
  * 流动性池：SOL-USDT池子
  * 你想买SOL，直接从池子里换
  * 价格由算法自动计算
  * 不需要等待买家/卖家

**例子：**
你要买100 SOL：
1. 连接钱包
2. 输入USDT数量
3. 智能合约自动从流动性池扣USDT、给你SOL
4. 完成


✅ 不需要KYC（匿名）
✅ 不会跑路（智能合约控制）
✅ 新币秒上线（任何人都能创建交易对）
✅ 你控制资金（钱在你钱包）


⚠️ 流动性可能不足（买卖有滑点）
⚠️ 价格波动大
⚠️ 可能买到Rug Pull的骗局币
⚠️ Gas费（每笔交易都要付）

**Mia总结：**
CEX = 正规商店（安全但慢）
DEX = 自由市场（快但野）
