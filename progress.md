# 🎯 我的学习进度

欢迎来到你的学习进度追踪页面！这里会自动记录你在90天学习计划中的所有进度。

<ProgressTracker />

---

## 📊 如何使用进度追踪？

### ✅ 记录进度

在每一天的任务页面中，点击checkbox即可标记完成：

- ✅ 完成的任务会显示为打勾状态
- 📊 进度会自动保存在你的浏览器中
- 🔄 可以随时查看整体进度

### 📈 查看统计

在本页面你可以看到：

1. **总体进度** - 90天计划的整体完成情况
2. **已完成任务数** - 你已经完成了多少个任务
3. **当前天数** - 你已经学习了多少天
4. **各模块进度** - 每一天/每一周的详细进度

### 💾 导出进度

点击"导出进度"按钮，可以下载一个JSON文件，包含你的所有学习记录。

**用途：**
- 📋 备份你的学习进度
- 📊 分析你的学习数据
- 🎯 追踪学习效率

### 🔄 重置进度

如果你想重新开始，可以点击"重置进度"按钮清空所有记录。

::: warning 注意
重置操作不可恢复，请谨慎使用！建议先导出备份。
:::
